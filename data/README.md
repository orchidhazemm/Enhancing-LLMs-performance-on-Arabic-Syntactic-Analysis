# Data

Datasets used in this project are stored on Google Drive (too large for GitHub). Download and place them in this folder for training and evaluation.

## Datasets (download links)

| Dataset | Description | Link |
|--------|-------------|------|
| **Alpaca Arabic (cleaned)** | Instruction-tuning data, Arabic | [alpaca_arabic_clean.csv](https://drive.google.com/file/d/1keQsE__ASE1XJPBtUxIbDIGYkOYqQ6lF/view?usp=share_link) |
| **Beetlware** | Arabic corpus | [beetlware.csv](https://drive.google.com/file/d/1c53Usat9YczAbkNN2MDgrfElXAiOlFyc/view?usp=share_link) |
| **Gemma2 Arabic instruct (cleaned)** | Instruction data for Gemma2, Arabic | [gemma2_ar_instruct_cleaned.jsonl](https://drive.google.com/file/d/1GaRX2Hs_b4oBuOodUv_QVDf8J6IWe_ib/view?usp=share_link) |
| **ShareGPT Arabic instruction** | Instruction data, Arabic | [sharegpt_arabic_instruction.jsonl](https://drive.google.com/file/d/1KsaBhO7vPlBLp3o5hTxNQ_INkqsz09kW/view?usp=share_link) |

## Large files

Large files are not committed to Git (see root `.gitignore`). For other options (Git LFS, DVC), see the main [README](../README.md).
